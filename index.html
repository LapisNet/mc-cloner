ReadMe

How to contribute to this version repo
如何为本版本库做出贡献

本仓库为各位AndroidのMinecraft爱好者提供Mc不分版本的共存版本，方便各位玩家同时Play两个不同的版本，不用卸载其中之一，解决新版本没有假期创作者，正式版没有延迟渲染等Question!
如果你想为本版本库做出贡献，您可以向本仓库(LapisNet/mc-cloner)提交Issue，您只需要提供一段包含版本内容的JSON对象！
具体内容可参考 Version.json 喵
